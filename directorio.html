Directorio General BForward

Formulario para alta y edición de entidades, con roles, aliases, contactos y direcciones. Este HTML está preparado para enviar un JSON limpio a un webhook o backend.

Configuración

Si aún no tienes endpoint, el formulario solo mostrará el JSON. Cuando lo conectes, apunta el valor a tu webhook o API.

Recomendado: conectar este formulario a un webhook de n8n y desde ahí ejecutar SELECT directorio.guardar_entidad_completa($1::jsonb).

Entidad

Datos principales del registro maestro.

Roles logísticos y comerciales

Una misma entidad puede tener varios roles.

Aliases / nombres alternos

Útil para BL Name, Billing Name, nombre documental u otras variantes del partner.

Puedes capturar varios nombres alternos.

Contactos

Captura contactos operativos, de facturación o principales.

Puedes capturar tantos contactos como requieras.

Direcciones

Separadas por tipo: fiscal, billing, almacén, patio, entrega, oficina, etc.

Puedes capturar varias direcciones por entidad.

Vista previa del payload que se enviará.
